# PPCom 界面

### 客服团队名称、欢迎信息
当用户将鼠标悬停在网页右下角的PPCom图标上时，图标上方会显示一个 “悬浮卡片”，卡片显示了客服团队名称和欢迎信息，默认是：

    有什么可以帮忙的，一起聊两句吧？

客服管理员可以在PPConsole的 **团队设置** 里修改客服团队名称和欢迎信息。

### PPCom图标颜色
图标颜色指的是在网站右下角显示的PPCom聊天图标的颜色。

客服管理员可以在PPConsole的 **团队设置-用户界面** 里修改PPCom图标颜色。

### 客服组
客服管理员在PPConsole的 **团队设置-客服组** 里可以给客服团队添加客服组。

在PPCom的会话列表界面，团队的客服组显示在列表最上面。当用户点击一个客服组时，会话列表里会生成一个新会话，会话成员是该用户和该客服组里的所有客服。
